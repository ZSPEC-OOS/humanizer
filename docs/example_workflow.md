# Example Workflow

1. Start backend: `uvicorn backend.app:app --reload`
2. Start frontend: `cd frontend && npm install && npm run dev`
3. In **Train Framework**, paste AI and humanized pair and click Analyze Pair.
4. Review rules and update rule bank via API.
5. In **Convert**, submit new text with hybrid mode.
6. Inspect score cards, rule trace, and diff viewer.
